PROJECTS

PROJECT 4: Full-Stack Django and React App

FAM

Pair Coding

Focused on our favourite things, we built a Full-Stack online publication where users can browse creative content within Film, Art and Music. I built the back-end using Python and PostgreSQL using the Django REST framework. This was my first project working with Python and I’m very happy with the final product. Logged in users can curate favourites lists and leave ratings and reviews. Users can browse previous posts in the index pages or by individual medium in a more traditional show card view. I plan to integrate with a music API along with other added features.

PROJECT 3: Full-Stack MERN App

ACCOUNTABLE

Group

Accountable is an app for splitting expenses with your friends and helps keep track of who owes what. After a week’s tuition in node.js and Express, I took on the responsibility of developing the back-end, building our own RESTful API and testing my knowledge and understanding of embedded and referenced relationships. I went on to build the ‘add friend’ feature and ‘search for friend’ functionality. A big win from this project was my use case of Some, Every, Map and Filter to overcome the logic of only raising an expense with another user if they are a friend.

PROJECT 2: React Hackathon Integrating With Third Party API

CELEBRITY LOOKALIKE

Pair Coding

As a pair, we built a React.js app integrating with two third part APIs to build a celebrity look alike app. Users upload an image of themselves and the site responds with their celebrity look alike name and corresponding image. Working against a 48hour deadline we divided the responsibilities into front-end and back-end. I worked on the front-end and styling for this project.

PROJECT 1: JavaScript Fundamentals Grid Based Game

SPACE INFECTORS

Solo

I created my own version of 80s arcade game Space Invaders. Players must shoot at the invading alien army before they reach the planet's surface using the arrow keys and space bar to shoot. Watch out for the Mothership! This was a solo project and was essential for me in grasping the fundamentals of vanilla JavaScript, arrays and setIntervals. It also proved to me very early on the importance of clear planning and wireframing and I have since become familiar with many useful sites to help with this, including Figma, Balsamiq, Trello and Miro
This game took 8 days to build.
